## 02 웹캠 영상에서 에지 검출
- 웹캠을 사용하여 실시간 비디오 스트림을 가져온다.
- 각 프레임에서 Canny Edge Detection을 적용하여 에지를검출하고 원본 영상과 함께 출력한다.
